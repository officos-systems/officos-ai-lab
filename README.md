# OFFICOS AI LAB

社内3人（石川・中村・熊田）向けの共有ダッシュボード。1画面で以下を共有する。

- **出来てない事（課題）**: 投稿フォームから登録（投稿者・日付・進捗を記録）
- **出来た事（解決ログ）**: 検索可能なナレッジ（LLM Wiki方式で蓄積予定）
- **AIニュース**: RSS自動取得（Supabase Edge Function `ai-news` 経由）

## 構成

- `index.html` … ダッシュボード本体（静的HTML1枚）。データは Supabase（課題テーブル）＋ Edge Function（ニュース）をクライアントJSから利用。
- `start.bat` … 社内サーバーで静的配信（ポート5577）するための起動スクリプト。
- `README.txt` … 社内サーバーへの設置手順。

## 配信

社内の常時稼働サーバーで `start.bat` により静的配信し、社内ネット内のブラウザから閲覧する（ポート5577）。

## 連携予定

- セキュリティダッシュボード（C石）を「SECURITY」タブとして相乗り（agent-comms Issue #51）。
- 「出来た事」の自動蓄積（毎日18時のGit push、push前に機微情報フィルタ PAT-12 を適用）。
