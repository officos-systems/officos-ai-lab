OFFICOS AI LAB ― 社内サーバーで動かす手順
==========================================

このフォルダを、常時起動しているサーバーPCに置いて動かすと、
社内の3人がブラウザでいつでもダッシュボードを開けるようになります。
（データは Supabase クラウドに保存されるので、サーバーPCはネット接続が必要です）


■ セットアップ手順
--------------------------------------------
1. このフォルダ（officos-ailab-sample）を丸ごとサーバーPCにコピー
   例：USBメモリ／社内の共有フォルダ など

2. サーバーPCで「start.bat」をダブルクリック
   → 黒い画面が出て「起動します…」と表示されれば成功
   → この画面は開いたまま（閉じると停止します）

3. サーバーPCの IP アドレスを調べる
   スタート →「cmd」と入力して開く →「ipconfig」と入力
   →「IPv4 アドレス」の数字（例：192.168.0.50）がそれ

4. 3人は各自のブラウザで次を開く
   http://（上のIP）:5577/
   例：http://192.168.0.50:5577/
   ※ ブックマーク推奨


■ PC起動時に自動で立ち上げる（おすすめ・常時稼働向け）
--------------------------------------------
1. start.bat を右クリック →「ショートカットの作成」
2. Windowsキー + R →「shell:startup」と入力して Enter
   （スタートアップ フォルダが開きます）
3. 作ったショートカットを、そのフォルダに入れる
→ 次回からPC起動時に自動で立ち上がります


■ つながらない時（ファイアウォール）
--------------------------------------------
・初回に「アクセスを許可しますか？」が出たら「許可」を押す
・出ない／繋がらない場合は、サーバーPCで「管理者として」cmd を開き、
  次を1回だけ実行：
  netsh advfirewall firewall add rule name="OFFICOS AI LAB" dir=in action=allow protocol=TCP localport=5577


■ Python が無い場合
--------------------------------------------
・かんたん：python.org から Python を入れる
  （インストール画面で「Add Python to PATH」にチェック）
・または Docker がある場合は、このフォルダで次を実行（自動再起動つき＝常時稼働向き）：
  docker run -d --name ailab --restart unless-stopped -p 5577:80 -v "（このフォルダのフルパス）":/usr/share/nginx/html:ro nginx:alpine
